# Majestic-Sights

## サイト概要
·国々の雄大な名所についてのSNS

### サイトテーマ
·各国の魅力を知ろう

### テーマを選んだ理由
·色々な国に興味がありシェアをしあって国の魅力を認知できると考えた為

### ターゲットユーザ
·世界各国の魅力を知りたい貴方様へ

### 主な利用シーン
·各国のここに行ってみたい､各国のオススメスポットなどを知りたいとき
